Privacy Policy

Shelfish for macOS

Overview

Shelfish is a macOS utility that provides a drag-and-drop shelf at the edge of your screen. We are committed to protecting your privacy.

Data Collection

Shelfish does not collect, store, or transmit any personal data. Specifically:

  • No analytics or tracking of any kind
  • No network connections are made
  • No user data is sent to any server
  • No crash reports are collected

File Access

Shelfish temporarily holds references (URLs) to files that you drag onto the shelf. These references exist only in memory during the app session and are never written to disk or transmitted. No files are copied or modified.

Local Storage

Shelfish stores only your preference for the shelf edge position (left, right, top, or bottom) using macOS UserDefaults. No other data is persisted.

Third-Party Services

Shelfish does not integrate with or use any third-party services, SDKs, or frameworks that collect data.

Contact

If you have any questions about this privacy policy, please open an issue on our GitHub repository.
